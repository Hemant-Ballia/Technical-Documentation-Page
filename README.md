# 📘 Technical Documentation Page

A modern and responsive Technical Documentation webpage built using HTML5 and CSS3.  
This project demonstrates responsive layouts, fixed sidebar navigation, structured documentation sections, and clean UI design.

---

## 🚀 Features

- Responsive web design
- Fixed sidebar navigation
- Smooth section navigation
- Styled code blocks
- Clean documentation layout
- Media query support
- Beginner-friendly project
