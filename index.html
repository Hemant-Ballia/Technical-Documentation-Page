Introduction

JavaScript is a programming language used to build interactive websites.

It works together with HTML and CSS for web development.

console.log("Hello World");
  • Easy to learn
Variables

Variables are used to store data values.

JavaScript supports var, let, and const.

let name = "Hemant";
  • Use let for changeable values
Functions

Functions help organize reusable code blocks.

A function runs only when it is called.

function greet() {}
  • Functions improve code reusability
Loops

Loops repeat code multiple times.

Common loops are for, while, and do while.

for(let i = 0; i < 5; i++){}
  • Loops save time in coding
Arrays

Arrays store multiple values in one variable.

Arrays are useful for handling collections of data.

let fruits = ["Apple", "Mango"];
  • Arrays use index numbers
